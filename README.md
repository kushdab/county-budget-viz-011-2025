# County Budget Viz 011-2025

An R-based analysis suite designed to visualize annual budget allocations and spending patterns across the 47 counties of Kenya. This project provides insights into sectoral funding and spending efficiency (absorption rates).

## Features
- **Automated Data Processing**: Aggregates raw budget figures by county and sector.
- **Efficiency Metrics**: Calculates budget absorption rates per county.
- **Advanced Visualizations**: 
  - Scatter plots for budget vs. efficiency.
  - Stacked bar charts for sectoral breakdowns.
  - Comparative analysis of national averages.

## Requirements
- R version 4.0+
- Libraries: `tidyverse`, `scales`, `gridExtra`

## Usage
1. Clone the repository.
2. Open `analysis.R` in RStudio.
3. Run the script to generate summary statistics and 3 high-quality visualizations.

## Data Source
Currently uses simulated data based on 2024/2025 fiscal projections for demonstration purposes.
